소개

이 웹사이트는 온라인 플래너입니다. 오늘 할 목표 과제를 적으면 AI가 과제에 맞는 점수를 자동으로 할당해줍니다. 목표를 달성할수록 누적 점수만큼 캐릭터가 앞으로 전진하는 방식으로 동기부여를 만들어줍니다.

어떻게 동작하나요?

  • 사용자가 오늘의 목표 과제를 작성
  • AI가 과제 난이도/소요시간/중요도에 맞춰 점수 자동 할당
  • 목표를 완료 처리하면 점수가 누적
  • 누적 점수만큼 캐릭터가 앞으로 이동(진행도 시각화)

예시

오늘의 목표 과제
- 영어 단어 30개 암기 → 20점
- 수학 문제 10개 풀기 → 30점
- 운동 30분 → 15점

완료하면 총 65점 만큼 캐릭터 전진

홈으로 돌아가 오늘의 할 일을 적어보세요!
